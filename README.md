# 🚑 Emergency Hospital Bed Availability & Ambulance Management System

## 📖 Overview

The Emergency Hospital Bed Availability & Ambulance Management System is a web-based application designed to help ambulance drivers locate nearby hospitals with available beds during emergencies. The system provides real-time bed availability information, reducing patient response time and improving emergency healthcare services.

---

## 🎯 Objectives

- Find nearby hospitals during emergencies.
- Display real-time bed availability.
- Reduce ambulance response time.
- Improve hospital bed management.
- Assist patients in receiving timely treatment.

---

## ✨ Features

### Hospital Admin
- Secure Login
- Update hospital information
- Manage Normal, ICU, and Emergency bed availability

### Ambulance Driver
- Secure Login
- Search nearby hospitals
- View available bed status
- Open hospital location using Google Maps

### System Admin
- Manage hospitals
- Manage ambulance driver accounts
- View reports and system data
